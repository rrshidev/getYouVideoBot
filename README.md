# YouTube Video Downloader Bot

Telegram-бот для скачивания видео с YouTube с использованием yt-dlp.

## 🚀 Возможности

- Скачивание видео с YouTube в различных качествах (360p, 480p, 720p, 1080p)
- Автоматическое определение информации о видео
- Проверка размера файла (лимит 50MB для Telegram)
- Удобный интерфейс с кнопками выбора качества
- Автоматическое удаление временных файлов

## 📋 Требования

- Python 3.10+
- FFmpeg (для обработки видео)
- Доступ к YouTube (без ограничений)

## 🛠️ Установка

1. **Клонируйте репозиторий:**
   ```bash
   git clone <repository-url>
   cd getYouVideoBot
   ```

2. **Установите зависимости:**
   ```bash
   pip install -r requirements.txt
   ```

3. **Установите FFmpeg:**
   
   **Windows:**
   - Скачайте с [официального сайта](https://ffmpeg.org/download.html)
   - Добавьте в PATH или установите через winget: `winget install ffmpeg`
   
   **Linux (Ubuntu/Debian):**
   ```bash
   sudo apt update
   sudo apt install ffmpeg
   ```
   
   **macOS:**
   ```bash
   brew install ffmpeg
   ```

4. **Настройте переменные окружения:**
   
   Создайте файл `.env` и добавьте токен бота:
   ```
   BOT_TOKEN=your_telegram_bot_token_here
   ```

   Как получить токен бота:
   1. Найдите [@BotFather](https://t.me/botfather) в Telegram
   2. Отправьте команду `/newbot`
   3. Следуйте инструкциям и получите токен

## 🚀 Запуск

### Локальный запуск
1. **Запустите бота:**
   ```bash
   python main.py
   ```

2. **Проверьте работу:**
   - Найдите вашего бота в Telegram
   - Отправьте команду `/start`
   - Отправьте ссылку на YouTube видео

### 🐳 Деплой на сервер через Docker

#### Автоматический деплой
```bash
# Клонируйте репозиторий
git clone <repository-url>
cd getYouVideoBot

# Настройте .env файл
cp .env.example .env
# Добавьте BOT_TOKEN в .env

# Запустите деплой
chmod +x deploy.sh

```

#### Ручной деплой
```bash

# Проверка статуса
docker-compose ps

# Просмотр логов
docker-compose logs -f
```

- Скачивание видео может нарушать условия использования YouTube

## 🔧 Конфигурация

### Изменение качества по умолчанию

В файле `main.py` можно изменить настройки качества:

```python
# В методе download_video()
format_string = 'best[height<=720]+bestaudio/best[height<=720]'  # Измените 720 на нужное значение
```

### Настройка прокси (если необходимо)

Если у вас есть ограничения доступа к YouTube, добавьте в `ydl_opts`:

```python
self.ydl_opts = {
    # ... другие настройки
    'proxy': 'http://your-proxy:port',  # или socks5://your-proxy:port
}
```

## 🐛 Устранение проблем

### Проблема: "Не удалось скачать видео"

**Возможные причины:**
- Видео недоступно для скачивания
- Файл слишком большой (>50MB)
- Проблемы с доступом к YouTube

**Решения:**
- Попробуйте другое видео
- Проверьте доступ к YouTube
- Убедитесь, что FFmpeg установлен

### Проблема: "Файл слишком большой"

**Решение:**
- Выберите более низкое качество видео
- Для больших файлов можно настроить сжатие через FFmpeg

### Проблема: Бот не запускается

**Проверьте:**
- Правильность токена в `.env`
- Установлены ли все зависимости
- Доступен ли Python 3.10+
